# fix screentearing
create file /usr/share/X11/xorg.conf.d/20-intel.conf with content:
```
Section "Device"  
  Identifier  "Intel Graphics"  
  Driver      "intel"  
  Option      "AccelMethod"  "sna"  
  Option      "TearFree"     "true"  
EndSection
```
Display > Scale -> Custom 0.5, 0.5 (or whatever is best for your display size)

# setup window keybinds
Window Manager > Keyboard
- FIRSTLY clear ALL values you're changing, otherwise the config will fail to properly set new binds
- Cycle windows -> Alt+Tab
- Cycle windows (reverse) -> Shift+Alt+Tab
- Close window -> Shift+Super+Q
- Maximize window -> Super+F
- Hide window -> Super+Down
- Resize window -> Shift+Super+R
- Toggle fullscreen -> F11
- Move window to workspace n -> Shift+Super+n
- Tile window to the left/right -> Shift+Super+Left/Right
- Show desktop -> Super+D
- Left workspace -> Shift+Super+Tab
- Right workspace -> Super+Tab
- Workspace n -> Super+n
- LASTLY test ALL keybinds
- IF a keybind doesn't work, remove its duplicate in ~/.config/xfce4/xfconf/xfce-perchannel-xml/xfce4-keyboard-shortcuts.xml
Window Manager > Focus > Click to focus
Window Manager Tweaks > Accessibility > Automatically tile windows when moving towards the screen edge -> Enabled
